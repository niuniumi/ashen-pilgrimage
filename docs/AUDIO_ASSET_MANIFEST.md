# 音频素材与授权清单

更新日期：2026-07-11

产品音频位于 `public/assets/audio/v2/`。BGM 已统一为 48 kHz 立体声，并进行约 `-18 LUFS` 的响度标准化；高频效果音使用 2 至 3 个样本轮换，避免连续操作时机械重复。
