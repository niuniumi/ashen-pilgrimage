# 灰烬圣途 GDD

## 核心定位

《灰烬圣途》是一款中世纪幻想像素风 Roguelike Deckbuilder。玩家扮演余烬行者，穿过灰烬王国 Eldermark 的暮鸦村，击败第一章 Boss 无首守墓骑士。

## 世界观

Eldermark 曾由圣火教会庇护。年迈国王为永生献出心脏后，圣火变成灰白色并开始吞噬灵魂。王都被灰烬覆盖，骑士变成空壳，修士变成怪物，幸存者称灾难为“灰烬圣变”。

## 第一阶段范围

- 主菜单、指南、角色选择、地图、战斗和战后流程完整闭环。
- 地图包含普通战斗、精英、事件、商店、休息、宝箱和 Boss。
- 战斗包含抽牌、能量、护甲、敌人意图、状态、胜败结算。
- 使用 localStorage 保存旅途和设置。

## 角色

流亡骑士：攻击、连击、伤痕、爆发。伤痕可被撕裂并按层数造成额外伤害。

圣烛修女：防御、烛印、延迟伤害、削弱。烛印在敌人回合末燃烧。

灰血炼金师：自伤、药剂、腐蚀、低血量爆发。自伤换取力量、抽牌或高伤害。

## 战斗循环

1. 玩家回合开始，抽 5 张牌，能量恢复到 3。
2. 玩家点击卡牌，攻击牌再选择敌人，防御和技能牌直接生效。
3. 打牌消耗能量，使用后进入弃牌堆或消耗堆。
4. 结束回合后敌人按头顶意图行动。
5. 敌人全部死亡进入奖励或胜利结算，玩家死亡进入失败结算。

## 视觉方向

中世纪幻想、像素风、羊皮纸、黑铁、金边、篝火、烛火、古堡、墓园、旧地图。第一阶段使用 Phaser Graphics 程序化绘制，避免把参考图中的假 UI 或文字残留带入游戏。
