# 灰烬圣途 1.0 发布说明

发布日期：2026-07-10

本版本将原型升级为可完整通关的三章产品。重构覆盖局内随机、存档恢复、战斗职业循环、遭遇与首领阶段、地图、誓约、事件、结局、音频、键盘输入、视觉层和自动化验收。

## 验收基线

- Node 单元测试覆盖随机、迁移、断点、遗物池、职业资源、遭遇、首领、地图、誓约、事件、结局、音频、输入、召唤与 QA 桥。
- 自动战斗模拟覆盖 3 个角色、3 种战斗类型、180 场战斗。
- 角色矩阵覆盖 3 个角色的选择、续局、牌池和战斗截图。
- 发布流程覆盖主菜单、序章、角色选择、誓约、三章地图、普通/精英/首领战、奖励、商店、事件、休息、宝箱、图鉴、暂停和胜负结算。
- 发布截图覆盖 `1920x1080`、`1536x864`、`1366x768` 和 `1280x720`。

## 兼容性

- v1 存档会迁移到 v2。
- 无有效战斗断点的旧版活动节点会自动回滚为可选节点。
- 9 个旧版遗物 ID 保持可加载，但不会进入新奖励和商店。

## 已知非阻塞项

- Phaser 与游戏逻辑仍在单个 Vite 主包中，构建会显示超过 500 kB 的体积提示。
- 当前交付为本地生产构建，未在本轮执行线上部署。
