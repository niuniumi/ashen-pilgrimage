# v2.1.0 Curated Actors

## 章节恢复

- 修复旧版 Boss 战斗断点缺少 `battleType` 时被当作普通战斗结算，导致第一章终点返回锁死地图的问题。
- 恢复时依次从战斗断点、待结算阶段和活动地图节点推断战斗类型。
- 新增真实点击章节回归：进入第一章 Boss、模拟旧存档刷新、击杀、章节结算、选誓约并验证第二章可选节点。

## 角色资产

- 28 种敌人全部切换为语义化独立 PNG 绑定，不再使用笼统的共享图集帧。
- 每个敌人记录源图真实朝向，在战斗中统一面向左侧玩家。
- 乌鸦信使和瘟疫鼠群更换为授权明确的 CC0 位图资产，不再使用程序化几何图形。
- 圣烛修女改为手持烛杖的独立主角形象；灰纱副院长保留四烛冠与圣匣敌人形象，两者不再共用图片。
- 失败页改用透明化、调色板量化的高精度墓碑资产，移除绿幕背景和简化几何墓碑。

## 质量门禁

- 新增 28 种敌人的真实战斗轮换测试，逐个检查资产 ID、敌人 ID、翻转状态、显示尺寸和页面错误。
- 新增敌人全名单截图报告和第一章到第二章的专项回归报告。
- 资产审计改为检查 PNG 结构与尺寸，实机测试负责检查非空和战场可见性。
