# v2.2 左向战斗资产与窄屏体验修复

## 视觉资产

- 三名主角改用同一张生成设定板拆分的 v3 透明 PNG，统一像素密度、光源、站高和右向战斗姿态。
- 圣烛修女取消头顶烛冠，改为单烛灯杖与银色圣徽；敌方灰纱副院长保留四烛冠，角色语义彻底分离。
- 24 种敌人（23 种人形与乌鸦信使）重建为明确的左向三分之二战斗姿态。
- 瘟疫鼠群替换为三只暗色病鼠组成的左向资产，修复粉彩画风、混合朝向和异常放大。
- 所有敌人生产 PNG 和资产元数据统一标记为左向，玩家角色统一为右向。

## 界面与交互

- 角色选择页默认真实选中流亡骑士，开始按钮与右侧详情不再出现伪选择状态。
- 悬停其他角色后离开会恢复已选角色详情。
- 起始牌组从三个微型格子合并为单行摘要，提高窄屏可读性。
- 战斗教程改为紧凑双按钮流程，增加六段进度，移除重复的关闭操作。

## 回归验证

- 新增 `qa:responsive-facing`，覆盖 1150×768 与 1171×731 两个用户问题视口。
- 28 种敌人逐个进入真实战斗场景验证语义 ID、生产纹理、朝向、尺寸和截图。
- 角色资产、鼠群比例和 v3 敌人绑定纳入单元测试，避免旧图重新进入生产。
