# v2.4.0 本地验收版说明

本版本在用户确认后才会部署，当前仅用于本地试玩验收。

## 视觉与资产

- 三名可玩角色统一为 300 像素视觉高度和同一落脚线，选中状态不再改变立绘比例。
- 70 张卡牌全部绑定独立语义插画帧，并按骑士、修女、炼金师、通用、状态和诅咒建立统一限色体系。
- 地图节点和通用 UI 切换为 20 帧独立像素图标图集，普通战斗、精英、事件、商店、休息、宝箱和首领不再共用图案。
- 新增可选节点呼吸光、悬停抬升和复位动效，并遵守“关闭动画”设置。
- 新增 Game-icons.net CC BY 3.0 原始矢量、逐项作者清单和可复现图集生成脚本。

## 交互与音频

- 角色选择支持方向键、A/D、数字 1-3、Enter/Space 和 Escape，鼠标与键盘状态保持同步。
- 修复角色选择场景重复进入后的锁定状态和旧输入监听残留。
- 音效按 UI、卡牌、战斗、首领和结算建立分层混音，增加轻微音高变化、共享悬停冷却和瞬态 BGM 压低。
- 修复连续重击或结算音效叠加时 BGM 过早恢复的问题。

## 质量门禁

- 增加角色选择生命周期、多分辨率落脚线、卡牌/图标覆盖、浏览器音频解码和节点动效合同测试。
- 修复多个浏览器 QA 工具写死预览端口的问题。
- 验证第一章首领结算可进入第二章，存档可恢复到奖励、事件、休息等阶段。
